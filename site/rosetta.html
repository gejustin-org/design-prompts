DNA

What makes Rosetta, Rosetta. Six principles encoded into every decision.

1

Dark Teal, Not Blue

#04191B is the signature. Sophisticated, distinctive, avoids "generic SaaS blue."

2

Noi Grotesk + Alternates

font-feature-settings: "ss03" on, "ss11" on gives subtle character.

3

Shadow-First Elevation

Soft shadows over heavy borders. Cards float, not sit.

4

4px Grid Discipline

Everything snaps to 4px. Spacing: 2/4/8/12/16/20/24/32/48/64.

5

Semantic Color Restraint

Color means something. Don't use color decoratively.

6

White Space as Feature

Generous padding and margins. The interface breathes.

Do / Don't

Guidelines for correct Rosetta usage.

✓ Do

  • Wrap button text in <ButtonText>, icons in <RosettaLeftIcon>/<RosettaRightIcon>
  • Use data-size/data-status attributes for sizing and status
  • Use aria-label on icon-only buttons and inputs without visible labels
  • Use Rosetta tokens for ALL colors, spacing, and radii
  • Use <Dismiss> wrapper for elements that close modals
  • Buttons for actions, <Link> for navigation
  • Maintain minimum 44px touch target

✕ Don't

  • Hardcode colors, spacing, or font sizes
  • Invent new tokens or components not in Rosetta
  • Use color decoratively—color carries semantic meaning
  • Put too much in modals—if scrolling needed, use a page
  • Use raw HTML <select>/<input>—use Rosetta fields
  • Skip labels on fields (use aria-label if visually hidden)
  • Use inverted button variants or --deprecated tokens

Accessibility

WCAG AA compliance is a hard requirement, not a nice-to-have.

Requirements

✓
Color Contrast—WCAG AA minimum. 4.5:1 for normal text, 3:1 for large text (18px+ or 14px bold).
✓
Focus Indicators—outline: var(--rosetta-interactive-focusOutline) solid 2px; outline-offset: 2px
✓
Keyboard Navigation—Enter/Space activate buttons. Escape closes modals. Tab navigates through focusable elements.
✓
Modal Accessibility—Automatic focus trap. aria-label required on every modal.
✓
Form Labels—Every field needs a label. Use aria-label for visually hidden labels. Focus moves to first error on submit.
✓
Table Semantics—<th> with scope, <caption> for table title, aria-sort on sortable headers.
✓
Reduced Motion—Respect prefers-reduced-motion: reduce. All animations must have a reduced-motion fallback.

Keyboard Patterns

KeyAction
TabNavigate to next focusable element
Shift+TabNavigate to previous focusable element
Enter / SpaceActivate buttons, toggle checkboxes
EscapeClose modals, dismiss toasts, cancel actions
