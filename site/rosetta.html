Design Philosophy

Trustworthy

Students trust this platform with their career. The design must feel reliable, not experimental.

Approachable

Despite handling serious workflows, it never feels cold or corporate. Warmth through typography and spacing.

Clear

Information hierarchy is paramount. Users scan, not read. Every visual decision aids comprehension.

Typography

Three font families: Noi Grotesk for UI (Inter fallback), Sansplomb for brand (Bebas Neue fallback), IBM Plex Mono for code.

font-sans
Noi Grotesk

Geometric sans-serif with humanist touches. Using Inter as web fallback.

font-brand
SANSPLOMB

Bold condensed for marketing and brand moments. Using Bebas Neue as web fallback.

font-mono
IBM Plex Mono

Clean monospace for code and technical labels.

Do's and Don'ts

Do

  • +Use semantic colors for meaning
  • +Rely on shadows for elevation
  • +Use generous whitespace
  • +Maintain the 4px grid
  • +Keep animations functional

Don't

  • -Use color decoratively
  • -Add playful or whimsical elements
  • -Create dark or moody interfaces
  • -Add decorative or ornate elements
  • -Use trendy or experimental patterns
