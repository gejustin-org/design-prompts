// born 1997 - 2012

chaotic good vibes Gen Z

Authenticity over polish, chaos over conformity, dark mode supremacy. This aesthetic lives in the tension between Y2K nostalgia and post-ironic internet culture. Design that grew up on Discord servers, Instagram stories, and For You pages.

// design philosophy

no cap

"Chaotic Good Vibes" - this design system channels the visual language of the generation raised on TikTok, meme culture, and digital-native aesthetics. It's anti-corporate, anti-minimalism, and proudly unfiltered. Where intentional "ugliness" is a flex, gradient meshes collide with chaotic layouts, and nothing is too weird to ship.

valid slay based real
  • Dark Mode Supremacy - Deep blacks are the canvas. Light mode is an afterthought. The sun is a deadly laser.
  • Neon on Void - Lime, hot pink, electric blue - saturated neon that pops like LED strips in a bedroom setup.
  • Gradient Mesh Chaos - Amorphous organic blobs that float, overlap, and create depth without traditional shadows.
  • Y2K Revival - Glossy surfaces, 3D floating elements, chrome effects, that early-2000s digital optimism.
  • Sticker Aesthetic - Elements that could be peeled off - bordered, rotated, overlapping, with "slapped on" energy.
  • Anti-Grid Grid - Bento layouts that organize chaos. Asymmetric but intentional. Breaking rules by following different ones.
  • Meme-Ready - Design that doesn't take itself too seriously. Room for personality and cultural references.

// the bold factor

mandatory

Gradient Mesh Backgrounds

Every page needs animated/static gradient mesh blobs. Not subtle - visible, colorful, alive.

3D Floating Objects

Decorative shapes floating in space with soft shadows and blur. Can be CSS or actual 3D renders.

Intentional "Ugly" Design

At least one element breaking rules - clashing colors, hard rotation, meme-adjacent styling. Embrace "no cap, this slaps" energy.

Mixed Typography Weights

Hero headlines must mix weights. Single-weight headlines are boring and mid.

Sticker Overlays

Rotated badges, hard shadows, slightly overlapping content. That "slapped on" energy.

Neon Glow Everything

Interactive elements MUST glow on hover. Not soft shadows - full neon glow with color.

Glassmorphism Cards

Frosted glass effects with backdrop blur. The aesthetic lives in transparency.

Y2K Chrome/Gloss Elements

At least one glossy/chrome effect - gradient overlays, reflections, or metallic surfaces.

Bento Grid Somewhere

At least one section uses true bento layout with intentionally mixed cell sizes.

Cultural References

Subtle nods to internet culture in copy, icons, or interaction patterns. Meme-ready spaces.
