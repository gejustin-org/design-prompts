// born 1997 - 2012

this is Gen Z

TikTok-native chaos meets anti-minimalism. Gradient meshes bleeding into the void, dark mode supremacy, and intentional ugliness that somehow slaps. No cap, this design system is built different.

// design philosophy

no cap

"Chaotic Good Vibes" - authenticity over polish, chaos over conformity, dark mode supremacy. This aesthetic lives in the tension between Y2K nostalgia and post-ironic internet culture. It's design that grew up on Discord servers, Instagram stories, and For You pages.

valid slay based
  • Dark Mode Supremacy - Light mode is an afterthought. The sun is a deadly laser.
  • Neon on Void - Saturated colors that pop like LED strips in a bedroom setup
  • Gradient Mesh Chaos - Amorphous blobs that float and breathe
  • Y2K Revival - Glossy surfaces, chrome effects, digital optimism
  • Sticker Aesthetic - Rotated, overlapping, that "slapped on" energy
  • Anti-Grid Grid - Bento layouts that organize chaos intentionally

// the bold factor

mandatory

Gradient Mesh Backgrounds

Every page needs animated/static gradient mesh blobs. Not subtle - visible, colorful, alive.

3D Floating Objects

Decorative shapes floating in space with soft shadows and blur. Can be CSS or actual 3D renders.

Intentional "Ugly" Design

At least one element that breaks rules - clashing colors, hard rotation, chaotic energy.

Mixed Typography Weights

Hero headlines must mix font weights. Single-weight headlines are boring and mid.

Sticker Overlays

Rotated badges, hard shadows, slightly overlapping content. That "slapped on" energy.

Neon Glow Everything

Interactive elements MUST glow on hover. Not soft shadows - full neon glow with color.

Glassmorphism Cards

Frosted glass effects with backdrop blur. The aesthetic lives in transparency.

Y2K Chrome Elements

At least one glossy/chrome effect - gradient overlays, reflections, or metallic text.
