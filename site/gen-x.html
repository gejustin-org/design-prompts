Philosophy

The Slacker Era Aesthetic

Whatever, Man - Authentic Apathy Meets DIY Rebellion

This is the aesthetic of the generation that grew up with MTV, watched the Berlin Wall fall, and came of age in the shadow of Boomer idealism. Gen X design captures the cynical independence, grunge authenticity, and early digital frontier of 1985-1999.

It's the visual language of zines, mixtapes, dial-up modems, and the belief that trying too hard is the ultimate sin.

Core Aesthetic DNA

  • Visual Language: Deliberately imperfect, anti-corporate, analog-meets-digital. Nothing is polished. Edges are rough.
  • Emotional Tone: Ironic detachment masking genuine passion. Simultaneously cynical and sincere.
  • Sound Design (Visual): VHS rewinding, dial-up handshake, Sonic Youth feedback, cassette deck click-clunk.

Design Pillars

The Zine Aesthetic

Photocopied textures, cut-and-paste collage, ransom-note typography, hand-drawn elements mixed with early desktop publishing.

VHS/CRT Artifacts

Tracking lines, static noise, RGB bleeding, the warm glow of a dying CRT monitor in a dark basement.

Grunge Typography

Distressed fonts, typewriter keys, early Mac system fonts, mixed weights and sizes that break every rule.

"Under Construction" Web 1.0

Animated GIFs, tiled backgrounds, visitor counters, horizontal rules, GeoCities chaos.

DIY Ethos

Everything looks handmade. Sticker bombing, spray paint textures, duct tape borders, skateboard graphics.

Cynical Color Palette

Muted earth tones punctuated by aggressive neon pops - like finding a highlighter in a thrift store army jacket.

The Anti-Patterns

  • Not Corporate: Actively mocks polished professionalism
  • Not Optimistic: Hope expressed through irony
  • Not Minimal: Layers of texture, collage, and visual noise
  • Not Pixel-Perfect: Intentional roughness as feature, not bug
  • Not Trying Too Hard: The moment it looks effortful, it's lost the plot

Typography

Breaking Every Rule

A deliberate collision of eras and contexts. Grunge display fonts for impact, typewriter mono for authenticity, and early Macintosh system fonts for dial-up nostalgia. Mix serifs with sans, ALL CAPS with lowercase in the same line.

xterm - fonts.conf

// Primary - Typewriter Authenticity
"Courier New", monospace
// Display - 90s Flyer Impact
IMPACT, HAETTENSCHWEILER
// Terminal - Matrix Mode
VT323 - DIAL-UP NOSTALGIA
// Body/UI - BBS Aesthetic
"Monaco", "Lucida Console", monospace

Text Effects

  • Xerox Blur: filter: blur(0.3px) - subtle photocopy degradation
  • VHS Chromatic Shift: text-shadow with RGB offset (-1px 0 #ff0000, 1px 0 #00ffff)
  • Typewriter Strike: Double-struck via layered text with slight offset
  • Glitch Text: CSS animation that randomly shifts position/color
Signature Elements

The Bold Factor

Mandatory elements that make this feel authentically Gen X:

  • VHS Tracking Artifacts - global overlay with horizontal distortion lines
  • Static Noise Texture - low-opacity SVG feTurbulence filter
  • "Under Construction" Element - ironic or sincere 90s web reference
  • Visitor Counter - mock badge ("You are visitor #000,000,042")
  • Mixed Type Treatments - CAPS and lowercase, different fonts in same line
  • Photocopy Degradation - blur and contrast boost simulation
  • Duct Tape / Collage Borders - irregular, handmade-feeling treatments
  • CRT Glow Effect - phosphor green glow on dark backgrounds
  • Easter Eggs on Hover - hidden messages, glitch reveals
  • Horizontal Rules (HR Tags) - construction stripes, dashed lines
  • 90s Beveled Web Buttons - outset/inset border-style
  • Terminal/Command Prompt Cards - window chrome with controls
Effects

Glitchy, Analog, Imperfect

Animations should feel like they're fighting against aging hardware. Quick jitters, tracking errors, and the occasional "freeze" followed by movement.

effects.css
_ [] x

/* CRT glow */

--shadow-crt: 0 0 20px rgba(0,255,0,0.15),

inset 0 0 60px rgba(0,255,0,0.03);


/* Xerox shadow (offset, not glow) */

--shadow-xerox: 3px 3px 0 rgba(0,0,0,0.3);


/* VHS tracking artifact */

--shadow-vhs: 0 2px 0 rgba(255,255,255,0.05),

0 -2px 0 rgba(0,0,0,0.3);


/* Chromatic aberration */

text-shadow: -1px 0 #ff0000, 1px 0 #00ffff;


/* Transition timing - slightly jerky */

transition: all 150ms steps(3);


/* VHS glitch timing */

transition: all 50ms cubic-bezier(0.68,-0.55,0.265,1.55);

Hover Behaviors

  • Buttons: Slight jitter, glow intensification, color inversion
  • Cards: Subtle lift with VHS artifact flash
  • Links: Underline appears with chromatic aberration
  • Images: Static noise overlay intensifies
